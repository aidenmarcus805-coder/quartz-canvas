# Apple UI

Use this skill when the selected desktop surface should feel refined, quiet, and native-like.

Prioritize:
- Stable panes and inspectors with clear ownership.
- Polished interaction states, keyboard focus, and disabled states.
- Concise text and high-quality spacing.
- Direct manipulation patterns where the selected workflow calls for them.

Avoid:
- Generic glass effects.
- Marketing-style hero sections inside product workspaces.
- Imitating branded assets, icons, or exact system app layouts.
